# Changelog

## 2.0.0 — Obscura MVP

- Added hosted Gemma note generation through OpenRouter.
- Added hosted Whisper transcription through Groq.
- Added MedGemma 4B GGUF as an optional local model.
- Replaced clinical note prompts with strict transcript-grounding rules.
- Added prominent local/hosted processing status and unverified-draft labeling.
- Focused the demo workflow on recording, transcription, note generation, and review.
- Rebranded the application, packages, storage, and desktop build as Obscura.
