# Correspondence Generation

Generate patient letters from clinical notes using templates and LLM refinement.

## Usage

1. Click "Generate Correspondence" after creating a note
2. Select a template or use custom instructions
3. Edit or refine the letter using LLM suggestions
4. Copy or save the final letter

## Input Modes

- **Ambient Listening:** Generate a letter based on the current clinical note derived from a recorded encounter
- **Dictate Mode:** Dictate the letter content directly, and the LLM will format it into a proper correspondence letter

## Features

- Templates for standardized letter formats (including a built-in "Dictation" template)
- Interactive refinement with LLM
- Automatic token management for context length
- Save letters for reference

## Templates

Create and manage letter templates in Settings:

- Default templates included
- Custom templates with specific instructions
- Set default template for quick generation
